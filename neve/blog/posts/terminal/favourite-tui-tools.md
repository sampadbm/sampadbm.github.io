### File Management & Navigation

- *rovr* — a terminal file manager
- *clipse* — clipboard manager in the terminal

### Editing & Writing

- *dawn* — a markdown editor
- *mcat* — cat on steroids, with support for all filetypes

### Presentations

- *presenterm / kyma* — markdown-based presentations with LaTeX/Typst support

### Bibliography

- *cobib / papis / bibiman / bibman* — terminal bibliography managers (Python-based)

### Development

- *crush* — terminal-based AI coding IDE by Charm (supports all models, like Cursor)

### Productivity

- *octotype / smassh* — typing trainers on the terminal
- *podliner / castero / musicpod* — listen to podcasts in the terminal
- *jotl* — battery monitor

### System & Networking

- *tuios* — terminal multiplexer OS
- *ssm* — simple SSH manager
- *wifitui* — WiFi manager
- *mdns-scanner* — mDNS network scanner
